# CODE WAY — онлайн школа программирования

<p align="center"><img src="logo/logo.png" alt="CODE WAY" width="220"></p>

Платформа для прохождения курсов программирования: видеоуроки, тесты с автопроверкой, домашние задания с проверкой преподавателем, прогресс обучения и сертификаты.

**Стек:** Django + Django REST Framework (JWT через djoser), SQLite/PostgreSQL, фронтенд — SPA на HTML/CSS/JS (дизайн-система построена по рекомендациям [ui-ux-pro-max](https://github.com/EvgeniyK96/ui-ux-pro-max-skill): стиль Flat Design + Accessible, фирменная палитра логотипа — navy `#16243D` + blue `#2563EB`, шрифт Inter).

---

## Возможности

| Функция | Реализация |
|---|---|
| Личный кабинет | Профиль, статистика, мои курсы, результаты тестов и ДЗ (`/api/users/dashboard/`) |
| Уроки с теорией | Markdown-теория, рендерится на фронтенде; курсы импортируются из `lessons/` |
| Практика в уроке | По типу курса (`Course.runner`): Python IDE (Pyodide/WebAssembly, поддержка `input()`), JavaScript-песочница, учебный мини-терминал (bash + git + docker, эмуляция) или SQL-консоль (настоящая SQLite через sql.js) |
| Тесты модулей | В конце каждого модуля тест (проходной 75%, у мини-курсов итоговый тест 70%); следующий модуль закрыт, пока тест не сдан — проверяется и на сервере |
| Домашние задания | Отправка текста/файла, статусы «на проверке / принято / на доработку», оценка и комментарий преподавателя |
| Сертификаты | Выдаются автоматически при 100% прохождении курса; публичная проверка по номеру |
| Прогресс обучения | Прогресс-бары по курсам, отметки пройденных уроков |

## Быстрый старт

```bash
source venv/bin/activate
python manage.py migrate
python manage.py import_courses         # все курсы из lessons/ (можно --only <slug>)
python manage.py createsuperuser        # либо используйте admin/admin12345, если уже создан
python manage.py runserver
```

Курсы: Python, C++, JavaScript, Java, DevOps (по 10 модулей с тестами) и мини-курсы Linux, Git, Docker, SQL (с итоговым тестом).

- Фронтенд: http://127.0.0.1:8000/
- Swagger: http://127.0.0.1:8000/swagger/
- Админка (создание курсов, проверка ДЗ): http://127.0.0.1:8000/admin/

Демо-учётки после `seed`: администратор `admin / admin12345`, преподаватель `teacher / teacher12345`.

## Как устроен процесс обучения

1. Студент регистрируется (`#/register`) и записывается на курс.
2. Читает теорию урока и сразу пробует код во встроенной IDE (кнопка «▶ Попробовать в IDE» у каждого примера), отмечает урок пройденным — растёт прогресс.
3. В конце модуля сдаёт тест: нужно набрать **минимум 75%**, иначе следующий модуль остаётся закрытым (блокировка проверяется на сервере, а не только в интерфейсе).
4. Отправляет домашние задания; преподаватель проверяет их в админке (статус «Принято» + оценка).
5. Когда пройдены все уроки, сданы все тесты и приняты все ДЗ — автоматически выдаётся сертификат с уникальным номером. Проверка подлинности: `GET /api/certificates/verify/<номер>/`.

## Основные эндпоинты API

| Метод | URL | Описание |
|---|---|---|
| POST | `/api/auth/users/` | Регистрация |
| POST | `/api/auth/jwt/create/` | Вход (JWT) |
| GET/PATCH | `/api/users/me/` | Профиль |
| GET | `/api/users/dashboard/` | Сводка личного кабинета |
| GET | `/api/courses/` | Каталог курсов (`?search=`, `?level=`) |
| POST | `/api/courses/<id>/enroll/` | Записаться на курс |
| GET | `/api/courses/lessons/<id>/` | Урок (только для записанных) |
| POST | `/api/courses/lessons/<id>/complete/` | Отметить урок пройденным |
| GET | `/api/assessments/quizzes/<id>/` | Тест (без правильных ответов) |
| POST | `/api/assessments/quizzes/<id>/submit/` | Сдать тест `{"answers": {"<question_id>": <choice_id>}}` |
| POST | `/api/assessments/homeworks/<id>/submit/` | Сдать ДЗ (текст и/или файл) |
| GET | `/api/certificates/` | Мои сертификаты |
| GET | `/api/certificates/verify/<номер>/` | Публичная проверка сертификата |

## Структура

```
apps/
├── users/          # профиль, дашборд личного кабинета
├── courses/        # курсы, модули, уроки, записи, прогресс (+ команда seed)
├── assessments/    # тесты, вопросы, попытки; ДЗ и их проверка
└── certificates/   # сертификаты + сервис автоматической выдачи
core/               # настройки (django-environ), маршруты, JWT, Swagger
templates/index.html + static/   # SPA-фронтенд
```
