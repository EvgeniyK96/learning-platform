## Урок 3.1. Условия if / else if / else, switch
**Время: 45 минут**

### Цель урока
Научиться управлять логикой программы в зависимости от условий с помощью `if/else` и `switch`.

### 1. Базовая конструкция if

```javascript
let age = 20;

if (age >= 18) {
    console.log("Доступ разрешён");
}
```

В отличие от Python, блок кода в JavaScript обозначается фигурными скобками `{}`, а не отступами (хотя аккуратные отступы всё равно важны для читаемости).

### 2. if / else

```javascript
let age = 15;

if (age >= 18) {
    console.log("Доступ разрешён");
} else {
    console.log("Доступ запрещён");
}
```

### 3. if / else if / else

```javascript
let score = 75;
let grade;

if (score >= 90) {
    grade = "A";
} else if (score >= 75) {
    grade = "B";
} else if (score >= 60) {
    grade = "C";
} else {
    grade = "F";
}

console.log(`Твоя оценка: ${grade}`);
```

### 4. Тернарный оператор

```javascript
let age = 20;
let status = age >= 18 ? "взрослый" : "несовершеннолетний";
console.log(status);
```

### 5. Конструкция switch

`switch` удобна, когда нужно сравнить одну переменную с несколькими конкретными значениями:

```javascript
let day = "понедельник";

switch (day) {
    case "суббота":
    case "воскресенье":
        console.log("Выходной");
        break;
    case "понедельник":
        console.log("Начало рабочей недели");
        break;
    default:
        console.log("Обычный будний день");
}
```

Важно не забывать `break` после каждого `case` — без него выполнение «провалится» в следующий блок (fall-through), что часто становится источником ошибок:

```javascript
let x = 1;

switch (x) {
    case 1:
        console.log("Один");
        // забыли break!
    case 2:
        console.log("Два");
        break;
}
// Выведет и "Один", и "Два"
```

### 6. Типичные ошибки новичков

- Используют `=` вместо `==`/`===` внутри условия.
- Забывают `break` в `switch`.
- Путают `else if` с несколькими независимыми `if` — в отличие от `else if`, независимые `if` проверяются все подряд, даже если предыдущее условие было истинным.

### Резюме урока

- `if/else if/else` — основная конструкция для ветвления логики программы.
- `switch` удобен для сравнения одной переменной с несколькими конкретными значениями, но требует `break` в каждом `case`.
- Тернарный оператор `условие ? значение1 : значение2` подходит для простых однострочных условий.

## Урок 3.2. Цикл for
**Время: 40 минут**

### Цель урока
Освоить классический цикл `for` — самый распространённый способ организации повторяющихся действий в JavaScript.

### 1. Синтаксис цикла for

```javascript
for (let i = 0; i < 5; i++) {
    console.log(i);
}
// выведет: 0 1 2 3 4
```

Цикл `for` состоит из трёх частей, разделённых точкой с запятой:
1. **Инициализация** (`let i = 0`) — выполняется один раз перед началом цикла.
2. **Условие** (`i < 5`) — проверяется перед каждой итерацией.
3. **Шаг** (`i++`) — выполняется после каждой итерации.

### 2. Перебор массива через for

```javascript
let fruits = ["яблоко", "банан", "груша"];

for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}
```

### 3. Цикл for...of — упрощённый перебор

Для перебора элементов массива или строки часто используется более современный и короткий цикл `for...of`:

```javascript
let fruits = ["яблоко", "банан", "груша"];

for (let fruit of fruits) {
    console.log(fruit);
}
```

### 4. Цикл for...in — перебор ключей объекта

```javascript
let student = { name: "Алина", age: 20, course: "JS с нуля" };

for (let key in student) {
    console.log(key, "-", student[key]);
}
```

> Важно: `for...in` предназначен для перебора свойств объектов, а не элементов массива — для массивов предпочтительнее `for...of` или `for` с индексом.

### 5. Вложенные циклы for

```javascript
for (let i = 1; i <= 3; i++) {
    for (let j = 1; j <= 3; j++) {
        console.log(`i=${i}, j=${j}`);
    }
}
```

### 6. Практический пример: таблица умножения

```javascript
let number = Number(prompt("Введите число:"));

for (let i = 1; i <= 10; i++) {
    console.log(`${number} x ${i} = ${number * i}`);
}
```

### Резюме урока

- Цикл `for` состоит из инициализации, условия и шага, разделённых точкой с запятой.
- `for...of` — современный способ перебора элементов массива, `for...in` — перебора ключей объекта.
- Вложенные циклы `for` часто используются для работы с таблицами и матрицами.

## Урок 3.3. Циклы while и do...while
**Время: 35 минут**

### Цель урока
Научиться использовать циклы `while` и `do...while` для повторения действий, когда заранее неизвестно точное количество итераций.

### 1. Цикл while

```javascript
let count = 1;

while (count <= 5) {
    console.log(count);
    count++;
}
```

Цикл `while` выполняется, пока условие остаётся истинным. Важно не забывать изменять переменную внутри цикла — иначе он станет бесконечным.

### 2. Цикл do...while

Отличие `do...while` в том, что тело цикла выполняется **как минимум один раз**, даже если условие изначально ложно — проверка условия происходит после выполнения блока кода:

```javascript
let count = 10;

do {
    console.log(count);
    count++;
} while (count <= 5);
// выведет 10, хотя условие count <= 5 изначально ложно
```

### 3. Сравнение while и for

`while` удобнее применять, когда неизвестно точное число повторений заранее (например, когда цикл должен продолжаться, пока пользователь не введёт правильный ответ). `for` лучше подходит, когда число повторений известно заранее.

### 4. break и continue

```javascript
let count = 0;

while (true) {
    count++;
    if (count > 5) {
        break;    // прерывает цикл полностью
    }
    if (count % 2 === 0) {
        continue;   // пропускает оставшуюся часть текущей итерации
    }
    console.log(count);
}
```

### 5. Практический пример: игра «Угадай число»

```javascript
let secretNumber = 42;
let guess;

do {
    guess = Number(prompt("Угадай число от 1 до 100:"));
    if (guess < secretNumber) {
        console.log("Больше!");
    } else if (guess > secretNumber) {
        console.log("Меньше!");
    }
} while (guess !== secretNumber);

console.log("Ты угадал!");
```

### 6. Опасность бесконечных циклов

Если условие цикла никогда не станет ложным, программа зависнет — браузер обычно предлагает остановить выполнение скрипта. Важно всегда проверять, что условие цикла действительно может измениться.

### Резюме урока

- `while` проверяет условие перед выполнением тела цикла, `do...while` — после, гарантируя как минимум одно выполнение.
- `while` предпочтителен, когда количество повторений заранее неизвестно.
- Обязательно нужно предусмотреть изменение условия внутри цикла, чтобы избежать бесконечного выполнения.

## Урок 3.4. break, continue, вложенные циклы
**Время: 40 минут**

### Цель урока
Углубить понимание управляющих конструкций `break` и `continue`, научиться работать с вложенными циклами и метками (labels).

### 1. Повторение: break и continue

```javascript
for (let i = 1; i <= 10; i++) {
    if (i === 5) {
        break;      // полностью прерывает цикл
    }
    console.log(i);
}
// выведет: 1 2 3 4

for (let i = 1; i <= 10; i++) {
    if (i % 2 === 0) {
        continue;   // пропускает чётные числа
    }
    console.log(i);
}
// выведет только нечётные числа
```

### 2. break во вложенных циклах

По умолчанию `break` прерывает только тот цикл, в котором он написан напрямую:

```javascript
for (let i = 0; i < 3; i++) {
    for (let j = 0; j < 3; j++) {
        if (j === 1) {
            break;    // прервёт только внутренний цикл
        }
        console.log(i, j);
    }
}
```

### 3. Метки (labels) для управления вложенными циклами

Если нужно прервать сразу и внешний, и внутренний цикл, используются метки:

```javascript
outerLoop: for (let i = 0; i < 3; i++) {
    for (let j = 0; j < 3; j++) {
        if (j === 1) {
            break outerLoop;   // прерывает оба цикла
        }
        console.log(i, j);
    }
}
```

Метки применяются нечасто в реальном коде, но полезно знать об их существовании при работе со сложной вложенной логикой.

### 4. Практический пример: треугольник из звёздочек

```javascript
let height = 5;
let result = "";

for (let i = 1; i <= height; i++) {
    result += "*".repeat(i) + "\n";
}

console.log(result);
```

### 5. Практический пример: поиск простого числа

```javascript
function isPrime(number) {
    if (number < 2) return false;
    
    for (let i = 2; i < number; i++) {
        if (number % i === 0) {
            return false;
        }
    }
    return true;
}

console.log(isPrime(7));    // true
console.log(isPrime(10));   // false
```

### 6. Типичные ошибки новичков

- Ожидают, что `break` во внутреннем цикле прервёт и внешний цикл тоже.
- Забывают, что `continue` переходит к следующей итерации, а не завершает цикл.

### Резюме урока

- `break` прерывает цикл полностью, `continue` — переходит к следующей итерации.
- По умолчанию `break`/`continue` относятся только к ближайшему циклу; метки позволяют управлять внешними циклами явно.
- Вложенные циклы часто применяются для работы с двумерными структурами данных и генерации паттернов.

## Урок 3.5. Практика: решение задач с условиями и циклами
**Время: 65 минут**

### Цель урока
Закрепить материал модуля, решив несколько практических задач с использованием условий и циклов.

### 1. Задача 1. Определение чётности чисел от 1 до 20

```javascript
for (let number = 1; number <= 20; number++) {
    if (number % 2 === 0) {
        console.log(`${number} — чётное`);
    } else {
        console.log(`${number} — нечётное`);
    }
}
```

### 2. Задача 2. Сумма чисел от 1 до N

```javascript
let n = Number(prompt("Введите N:"));
let total = 0;

for (let i = 1; i <= n; i++) {
    total += i;
}

console.log(`Сумма чисел от 1 до ${n}: ${total}`);
```

### 3. Задача 3. Подсчёт гласных букв в слове

```javascript
let word = prompt("Введите слово:").toLowerCase();
let vowels = "аеёиоуыэюя";
let count = 0;

for (let letter of word) {
    if (vowels.includes(letter)) {
        count++;
    }
}

console.log(`Количество гласных: ${count}`);
```

### 4. Задача 4. FizzBuzz

```javascript
for (let number = 1; number <= 100; number++) {
    if (number % 3 === 0 && number % 5 === 0) {
        console.log("FizzBuzz");
    } else if (number % 3 === 0) {
        console.log("Fizz");
    } else if (number % 5 === 0) {
        console.log("Buzz");
    } else {
        console.log(number);
    }
}
```

### 5. Задача 5. Таблица умножения от 1 до 5

```javascript
for (let i = 1; i <= 5; i++) {
    let row = "";
    for (let j = 1; j <= 5; j++) {
        row += `${i * j}\t`;
    }
    console.log(row);
}
```

### 6. Как подходить к решению подобных задач

1. Прочитай условие и определи входные и выходные данные.
2. Определи, нужен ли цикл с известным числом повторений (`for`) или зависящий от условия (`while`).
3. Реши задачу пошагово словами перед написанием кода.
4. Проверь пограничные случаи (N = 0, пустая строка, отрицательные числа).

### Резюме урока

- Практика закрепляет понимание условий и циклов лучше, чем чтение теории.
- FizzBuzz — классическая задача, часто встречающаяся на технических собеседованиях в разных языках программирования.
- Разбор условия задачи на «входные данные → обработка → результат» — универсальный подход к решению практических задач.
