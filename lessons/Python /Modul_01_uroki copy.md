## Урок 1.1. Что такое Python и где он применяется
**Время: 30 минут**

### Цель урока
Понять, что такое Python, чем он отличается от других языков программирования, и увидеть реальные примеры его применения.

### 1. Что такое Python

Python — это язык программирования общего назначения, созданный Гвидо ван Россумом в конце 1980-х — начале 1990-х годов. Название языка происходит не от змеи, а от британского комедийного шоу «Monty Python's Flying Circus» — автор языка был его поклонником.

Главные особенности Python:

- **Простой и читаемый синтаксис.** Код на Python похож на обычный английский текст и структурируется отступами, а не фигурными скобками.
- **Интерпретируемый язык.** Код выполняется построчно специальной программой-интерпретатором, а не превращается заранее в машинный код (как в компилируемых языках вроде C++).
- **Динамическая типизация.** Не нужно заранее указывать тип переменной — Python определяет его сам.
- **Огромная экосистема библиотек.** Практически для любой задачи уже существует готовое решение.

### 2. Чем Python отличается от других языков

Сравним коротко вывод текста на экран в разных языках:

```python
# Python
print("Привет, мир!")
```

Для сравнения, в языке Java для этого же действия потребовалось бы объявить целый класс с методом `main`. Именно поэтому Python часто называют одним из самых дружелюбных языков для новичков — минимум формальностей, максимум фокуса на логике задачи.

### 3. Где применяется Python

- **Веб-разработка.** Такие фреймворки, как Django и Flask, позволяют быстро создавать сайты и веб-сервисы (например, Instagram изначально был написан на Django).
- **Анализ данных и Data Science.** Библиотеки Pandas, NumPy, Matplotlib используются для обработки и визуализации больших массивов данных.
- **Машинное обучение и искусственный интеллект.** TensorFlow, PyTorch, scikit-learn — стандарт индустрии в исследованиях и разработке ИИ.
- **Автоматизация и скрипты.** Python отлично подходит для написания небольших программ, автоматизирующих рутинные задачи: обработку файлов, отправку писем, парсинг сайтов.
- **Тестирование ПО.** Многие QA-инженеры используют Python для написания автотестов.
- **Игры и обучающие проекты.** Библиотека Pygame позволяет создавать простые 2D-игры.
- **Интернет вещей (IoT) и встраиваемые системы.** MicroPython используется для программирования микроконтроллеров.

### 4. Почему Python — хороший выбор для первого языка

- Быстрый старт: первую рабочую программу можно написать за несколько минут.
- Огромное сообщество: почти на любой вопрос уже есть ответ на форумах и в документации.
- Востребованность на рынке труда: Python стабильно входит в топ языков программирования по спросу у работодателей.
- Универсальность: знания Python пригодятся и в вебе, и в анализе данных, и в автоматизации.

### 5. Как устроен процесс написания программы

1. Ты пишешь код в текстовом редакторе или IDE (в файле с расширением `.py`).
2. Запускаешь файл — интерпретатор Python построчно читает и выполняет код.
3. Видишь результат в терминале (консоли).

Это отличается, например, от компилируемых языков, где сначала нужно «собрать» программу в отдельный исполняемый файл, и только потом её запускать.

### Резюме урока

- Python — простой в изучении, но очень мощный язык программирования.
- Он используется в вебе, анализе данных, машинном обучении, автоматизации, тестировании и других областях.
- Код на Python выполняется построчно интерпретатором, без этапа компиляции.
- Простой синтаксис и большое сообщество делают Python отличным выбором для первого языка программирования.

## Урок 1.2. Установка Python и настройка IDE (VS Code)
**Время: 40 минут**

### Цель урока
Установить Python на компьютер, установить и настроить редактор кода Visual Studio Code для комфортной разработки.

### 1. Установка Python

1. Перейди на официальный сайт python.org в раздел «Downloads».
2. Скачай последнюю стабильную версию Python для своей операционной системы (Windows, macOS или Linux).
3. Запусти установщик. **Важно для Windows:** обязательно отметь галочку **«Add Python to PATH»** перед установкой — это позволит запускать Python из любой папки через терминал.
4. Дождись завершения установки.

Проверить, что Python установлен, можно через терминал (командную строку):

```bash
python --version
```

Если всё установлено верно, ты увидишь номер версии, например `Python 3.12.4`.

> На macOS и Linux иногда нужно использовать команду `python3 --version` вместо `python --version`, так как в системе может быть предустановлена более старая версия Python 2.

### 2. Что такое IDE и зачем она нужна

IDE (Integrated Development Environment — интегрированная среда разработки) — это программа, которая объединяет в себе текстовый редактор, инструменты запуска кода, отладчик и подсветку синтаксиса. Работать в IDE значительно удобнее, чем в простом блокноте.

Для этого курса мы будем использовать **Visual Studio Code (VS Code)** — бесплатный, лёгкий и очень популярный редактор с отличной поддержкой Python.

### 3. Установка VS Code

1. Перейди на сайт code.visualstudio.com и скачай версию для своей операционной системы.
2. Установи программу, следуя инструкциям установщика.
3. Запусти VS Code.

### 4. Установка расширения Python

1. В VS Code открой раздел расширений (иконка с квадратиками на боковой панели, либо сочетание клавиш `Ctrl+Shift+X`).
2. В поиске введи «Python».
3. Установи официальное расширение от Microsoft (Python extension for Visual Studio Code).

Это расширение добавит подсветку синтаксиса, автодополнение кода, встроенный запуск файлов и отладчик.

### 5. Первый запуск кода в VS Code

1. Создай новую папку для проектов курса, например `python_course`.
2. Открой эту папку в VS Code через меню «File → Open Folder».
3. Создай новый файл с именем `test.py`.
4. Напиши одну строку кода:

```python
print("Всё работает!")
```

5. Нажми кнопку запуска (треугольник в правом верхнем углу) или сочетание клавиш `Ctrl+F5` (Windows/Linux) / `Cmd+F5` (macOS).
6. В нижней части экрана откроется терминал с результатом выполнения программы.

### 6. Полезные настройки VS Code для начала работы

- Включи автосохранение файлов: «File → Auto Save».
- Установи тему оформления по вкусу: «File → Preferences → Theme → Color Theme».
- Увеличь размер шрифта в настройках, если тебе так удобнее читать код.

### Резюме урока

- Python устанавливается с официального сайта python.org, важно отметить пункт «Add to PATH».
- VS Code — удобная и бесплатная среда разработки с отличной поддержкой Python через расширение.
- Проверить установку Python можно командой `python --version` в терминале.
- Файл с кодом на Python имеет расширение `.py` и запускается через терминал или кнопку запуска в IDE.

## Урок 1.3. Первая программа: Hello, World!
**Время: 30 минут**

### Цель урока
Написать и запустить свою первую программу на Python, разобраться с функцией `print()` и понять, что происходит «под капотом» при запуске файла.

### 1. Традиция «Hello, World!»

Практически в любом курсе программирования первая программа выводит на экран фразу «Hello, World!» («Привет, мир!»). Это простой способ убедиться, что всё окружение настроено правильно и код действительно выполняется.

```python
print("Hello, World!")
```

Запусти этот код в VS Code — в терминале должна появиться строка `Hello, World!`.

### 2. Функция print()

`print()` — это встроенная функция Python, которая выводит переданные ей данные на экран (в консоль/терминал).

```python
print("Привет!")
print("Меня зовут Python")
print(2 + 2)
```

Обрати внимание: то, что находится в круглых скобках после `print`, называется **аргументом функции**. Функция `print()` может принимать текст (обязательно в кавычках), числа (без кавычек) и даже результаты вычислений.

### 3. Вывод нескольких значений

Функцию `print()` можно вызывать с несколькими аргументами через запятую — они будут выведены через пробел:

```python
print("Мне", 19, "лет")     # Мне 19 лет
```

### 4. Одинарные и двойные кавычки

Для текстовых данных (строк) в Python можно использовать как одинарные `'...'`, так и двойные `"..."` кавычки — разницы в работе между ними нет:

```python
print('Привет')
print("Привет")
```

Это удобно, когда сам текст содержит кавычку одного из типов:

```python
print("Она сказала: 'Привет!'")
```

### 5. Что происходит при запуске файла

1. Ты сохраняешь код в файле с расширением `.py`, например `hello.py`.
2. При запуске файла интерпретатор Python читает его построчно, сверху вниз.
3. Каждая строка выполняется как отдельная инструкция.
4. Если в коде есть ошибка, выполнение прерывается на строке с ошибкой, и Python выводит сообщение об ошибке (traceback).

### 6. Первая ошибка — это нормально

Попробуй специально ввести ошибку, например убрать закрывающую кавычку:

```python
print("Привет!)
```

Python покажет `SyntaxError` — сообщение об ошибке синтаксиса. Умение читать такие сообщения — важный навык программиста: Python довольно точно указывает, в какой строке и что пошло не так.

### Резюме урока

- Функция `print()` выводит данные на экран и является одной из самых часто используемых функций в Python.
- Строки заключаются в одинарные или двойные кавычки.
- Через запятую в `print()` можно передавать сразу несколько значений.
- Ошибки при запуске кода — это нормальная часть процесса обучения, важно научиться их читать.

## Урок 1.4. Знакомство с терминалом и структурой проекта
**Время: 40 минут**

### Цель урока
Научиться пользоваться терминалом для базовых операций и понимать, как правильно организовать структуру простого Python-проекта.

### 1. Что такое терминал

Терминал (командная строка, консоль) — это текстовый интерфейс для взаимодействия с операционной системой через команды. Программисты используют терминал для запуска программ, установки библиотек, работы с файлами и системами контроля версий.

В Windows это может быть **PowerShell** или **Командная строка (cmd)**, в macOS и Linux — **Terminal**. В VS Code терминал встроен и открывается через меню «Terminal → New Terminal» или сочетанием клавиш `` Ctrl+` ``.

### 2. Базовые команды терминала

| Команда (Windows) | Команда (macOS/Linux) | Что делает |
|---|---|---|
| `cd папка` | `cd папка` | Перейти в папку |
| `cd ..` | `cd ..` | Перейти на уровень выше |
| `dir` | `ls` | Показать список файлов в папке |
| `mkdir имя` | `mkdir имя` | Создать новую папку |
| `cls` | `clear` | Очистить экран терминала |
| `python file.py` | `python3 file.py` | Запустить Python-файл |

### 3. Пример работы в терминале

```bash
mkdir python_course        # создаём папку проекта
cd python_course           # переходим в неё
```

Внутри этой папки создаём файл `main.py` (через VS Code) и запускаем его:

```bash
python main.py
```

### 4. Структура простого Python-проекта

Даже на старте важно приучать себя к аккуратной организации файлов. Минимальная структура учебного проекта может выглядеть так:

```
python_course/
│
├── main.py          # основной файл с кодом программы
├── notes.md          # заметки по теме (по желанию)
└── practice/          # папка с练习ными файлами
    ├── task1.py
    └── task2.py
```

По мере роста курса в проекте будут появляться дополнительные файлы: с функциями, классами, тестами. Уже на этом этапе полезно привыкать к тому, что не весь код должен находиться в одном файле.

### 5. Абсолютные и относительные пути

- **Абсолютный путь** — полный путь к файлу от корня диска, например `C:\Users\Aliya\python_course\main.py`.
- **Относительный путь** — путь относительно текущей папки, например `practice/task1.py`, если ты уже находишься в папке `python_course`.

В большинстве случаев в учебных проектах удобнее использовать относительные пути.

### 6. Полезные привычки при работе с терминалом

- Используй `Tab` для автодополнения имён файлов и папок — это экономит время и снижает число опечаток.
- Стрелки «вверх/вниз» в терминале позволяют быстро повторить ранее введённые команды.
- Не бойся ошибок в терминале — они почти всегда содержат подсказку, что пошло не так.

### Резюме урока

- Терминал — основной инструмент для запуска программ и управления файлами в разработке.
- Базовые команды (`cd`, `ls`/`dir`, `mkdir`) позволяют перемещаться по файловой системе и создавать папки.
- Аккуратная структура проекта с самого начала облегчает дальнейшую работу с кодом.
- Умение работать с относительными путями пригодится при подключении файлов друг к другу.
