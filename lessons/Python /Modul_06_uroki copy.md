## Урок 6.1. Строки: методы и форматирование (f-strings)
**Время: 45 минут**

### Цель урока
Изучить строку как структуру данных, освоить основные строковые методы и научиться форматировать вывод с помощью f-строк.

### 1. Строка как последовательность символов

Строка (`str`) в Python — это неизменяемая (immutable) последовательность символов. Как и к списку, к строке можно обращаться по индексу и делать срезы:

```python
text = "Python"
print(text[0])      # P
print(text[-1])     # n
print(text[0:3])    # Pyt
```

Важно: строки нельзя изменить «на месте» — любая операция, «меняющая» строку, на самом деле создаёт новую строку.

```python
text = "Python"
text[0] = "J"   # TypeError: 'str' object does not support item assignment
```

### 2. Основные строковые методы

| Метод | Действие |
|---|---|
| `.upper()` | переводит в верхний регистр |
| `.lower()` | переводит в нижний регистр |
| `.strip()` | убирает пробелы по краям |
| `.replace(a, b)` | заменяет все вхождения `a` на `b` |
| `.split(sep)` | разбивает строку на список по разделителю |
| `.join(список)` | собирает список в строку через разделитель |
| `.find(x)` | возвращает индекс первого вхождения `x` (или -1) |
| `.count(x)` | считает количество вхождений `x` |

```python
text = "  Привет, Мир!  "
print(text.strip().lower())         # привет, мир!
print(text.replace("Мир", "Python"))

words = "яблоко,банан,груша".split(",")
print(words)   # ['яблоко', 'банан', 'груша']

joined = "-".join(["2026", "07", "12"])
print(joined)   # 2026-07-12
```

### 3. Форматирование строк: f-строки

f-строки (f-strings) — современный и самый удобный способ вставки значений переменных и выражений в текст:

```python
name = "Алина"
age = 20
print(f"Меня зовут {name}, мне {age} лет")
```

Внутри f-строки можно писать любые выражения, включая вызовы функций:

```python
price = 1999.5
print(f"Цена со скидкой: {price * 0.9:.2f}")
```

Конструкция `:.2f` — это спецификатор форматирования, округляющий число до двух знаков после запятой.

### 4. Другие способы форматирования (для общего понимания)

```python
# Метод .format()
print("Меня зовут {}, мне {} лет".format(name, age))

# Оператор % (устаревший способ)
print("Меня зовут %s, мне %d лет" % (name, age))
```

На практике в современном коде рекомендуется использовать именно f-строки как самый читаемый и быстрый способ.

### 5. Многострочные строки

```python
message = """
Уважаемый студент,
поздравляем с успешным прохождением модуля!
"""
print(message)
```

### 6. Практический пример: чек из магазина

```python
items = [("Хлеб", 350), ("Молоко", 620), ("Яблоки", 1200)]
total = 0

for name, price in items:
    print(f"{name:<10}{price:>6} тг")
    total += price

print("-" * 20)
print(f"{'Итого':<10}{total:>6} тг")
```

### Резюме урока

- Строки в Python неизменяемы — любая «модификация» создаёт новую строку.
- Ключевые методы: `.strip()`, `.split()`, `.join()`, `.replace()`, `.upper()`/`.lower()`.
- f-строки — рекомендуемый способ форматирования и вставки переменных в текст.

## Урок 6.2. Основы регулярных выражений
**Время: 45 минут**

### Цель урока
Понять, зачем нужны регулярные выражения, и научиться использовать базовые функции модуля `re` для поиска и извлечения данных из текста.

### 1. Что такое регулярные выражения

Регулярное выражение (regex) — это специальный шаблон для поиска и обработки текста по определённым правилам. Регулярные выражения незаменимы, когда нужно найти в тексте данные определённого формата: email-адреса, номера телефонов, даты и т.д.

В Python для работы с регулярными выражениями используется встроенный модуль `re`:

```python
import re
```

### 2. Базовые специальные символы

| Символ | Значение |
|---|---|
| `.` | любой символ |
| `\d` | любая цифра |
| `\w` | любая буква, цифра или `_` |
| `\s` | любой пробельный символ |
| `+` | один или более повторений |
| `*` | ноль или более повторений |
| `?` | ноль или одно повторение |
| `^` | начало строки |
| `$` | конец строки |

### 3. Функция re.search()

Ищет первое совпадение шаблона в строке:

```python
import re

text = "Мой номер: 8-701-234-5678"
match = re.search(r"\d{1}-\d{3}-\d{3}-\d{4}", text)

if match:
    print("Найдено:", match.group())
```

### 4. Функция re.findall()

Возвращает список всех совпадений в строке:

```python
text = "Свяжитесь со мной: ivan@mail.com или anna@gmail.com"
emails = re.findall(r"[\w.]+@[\w.]+", text)
print(emails)   # ['ivan@mail.com', 'anna@gmail.com']
```

### 5. Функция re.sub()

Заменяет все найденные совпадения на указанную строку:

```python
text = "Телефон: 8-701-234-5678"
hidden = re.sub(r"\d", "*", text)
print(hidden)   # Телефон: *-***-***-****
```

### 6. Практический пример: проверка формата email

```python
import re

def is_valid_email(email):
    pattern = r"^[\w.]+@[\w.]+\.\w+$"
    return re.match(pattern, email) is not None

print(is_valid_email("test@mail.com"))    # True
print(is_valid_email("не email"))            # False
```

### Резюме урока

- Регулярные выражения — мощный инструмент для поиска и обработки текста по шаблону.
- Модуль `re` предоставляет функции `search()`, `findall()`, `sub()` для разных задач.
- Регулярные выражения особенно полезны при извлечении email-адресов, номеров телефонов и проверке формата данных.

## Урок 6.3. Чтение и запись файлов
**Время: 45 минут**

### Цель урока
Научиться открывать, читать и записывать текстовые файлы в Python, а также правильно закрывать файлы после работы с ними.

### 1. Открытие файла

Для работы с файлами используется встроенная функция `open()`:

```python
file = open("notes.txt", "r")   # r - режим чтения (read)
content = file.read()
file.close()
print(content)
```

### 2. Режимы открытия файла

| Режим | Значение |
|---|---|
| `"r"` | чтение (файл должен существовать) |
| `"w"` | запись (создаёт новый файл или перезаписывает существующий) |
| `"a"` | добавление в конец файла |
| `"r+"` | чтение и запись |

### 3. Конструкция with — правильный способ работы с файлами

Ручное закрытие файла через `.close()` легко забыть, особенно если в коде происходит ошибка до этой строки. Рекомендуемый способ — конструкция `with`, которая автоматически закрывает файл после выполнения блока кода:

```python
with open("notes.txt", "r") as file:
    content = file.read()

print(content)
# файл уже автоматически закрыт здесь
```

### 4. Запись в файл

```python
with open("notes.txt", "w") as file:
    file.write("Первая строка\n")
    file.write("Вторая строка\n")
```

> Обрати внимание: режим `"w"` полностью перезаписывает файл, если он уже существовал. Для добавления данных без удаления существующего содержимого нужно использовать режим `"a"`.

### 5. Построчное чтение файла

```python
with open("notes.txt", "r") as file:
    for line in file:
        print(line.strip())    # .strip() убирает символ переноса строки
```

Также можно прочитать все строки сразу в виде списка:

```python
with open("notes.txt", "r") as file:
    lines = file.readlines()

print(f"Количество строк: {len(lines)}")
```

### 6. Практический пример: список любимых фильмов

```python
movies = ["Начало", "Интерстеллар", "Матрица", "Довод", "Оппенгеймер"]

with open("movies.txt", "w", encoding="utf-8") as file:
    for movie in movies:
        file.write(movie + "\n")

with open("movies.txt", "r", encoding="utf-8") as file:
    lines = file.readlines()

print(f"В файле {len(lines)} фильмов")
```

> Параметр `encoding="utf-8"` рекомендуется указывать всегда при работе с текстом на кириллице, чтобы избежать проблем с кодировкой на разных операционных системах.

### Резюме урока

- Конструкция `with open(...) as file:` — рекомендуемый способ работы с файлами, так как автоматически их закрывает.
- Режимы `"r"`, `"w"`, `"a"` определяют, как именно программа взаимодействует с файлом.
- При работе с кириллицей стоит явно указывать `encoding="utf-8"`.

## Урок 6.4. Работа с CSV и JSON
**Время: 50 минут**

### Цель урока
Научиться сохранять и загружать структурированные данные в форматах CSV и JSON — самых распространённых форматах для обмена данными.

### 1. Что такое CSV

CSV (Comma-Separated Values) — текстовый формат для хранения табличных данных, где значения разделены запятыми (или другим разделителем). Часто используется для экспорта данных из Excel/Google Sheets.

```python
import csv

with open("students.csv", "w", newline="", encoding="utf-8") as file:
    writer = csv.writer(file)
    writer.writerow(["Имя", "Возраст", "Курс"])
    writer.writerow(["Алина", 20, "Python с нуля"])
    writer.writerow(["Данияр", 19, "Python с нуля"])
```

### 2. Чтение CSV-файла

```python
import csv

with open("students.csv", "r", encoding="utf-8") as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)
```

### 3. Работа с CSV через DictReader и DictWriter

Более удобный способ — использовать словари вместо простых списков:

```python
import csv

with open("students.csv", "r", encoding="utf-8") as file:
    reader = csv.DictReader(file)
    for row in reader:
        print(row["Имя"], row["Возраст"])
```

### 4. Что такое JSON

JSON (JavaScript Object Notation) — текстовый формат для хранения структурированных данных, очень похожий по структуре на словари и списки Python. JSON — стандартный формат для обмена данными между сервером и клиентом, а также при работе с большинством внешних API.

### 5. Сохранение данных в JSON

```python
import json

students = [
    {"name": "Алина", "age": 20},
    {"name": "Данияр", "age": 19}
]

with open("students.json", "w", encoding="utf-8") as file:
    json.dump(students, file, ensure_ascii=False, indent=4)
```

Параметр `ensure_ascii=False` нужен, чтобы кириллица сохранялась в читаемом виде, а не в виде escape-последовательностей. Параметр `indent=4` делает файл красиво отформатированным.

### 6. Чтение данных из JSON

```python
import json

with open("students.json", "r", encoding="utf-8") as file:
    students = json.load(file)

for student in students:
    print(student["name"], student["age"])
```

### Резюме урока

- CSV удобен для табличных данных и совместим с Excel/Google Sheets.
- JSON — универсальный формат для структурированных данных, стандарт при работе с API.
- Модули `csv` и `json` входят в стандартную библиотеку Python и не требуют дополнительной установки.

## Урок 6.5. Исключения и обработка ошибок (try/except/finally)
**Время: 45 минут**

### Цель урока
Научиться перехватывать и обрабатывать ошибки, чтобы программа не «падала» при возникновении непредвиденных ситуаций.

### 1. Что такое исключение

Исключение (exception) — это ошибка, возникающая во время выполнения программы. Если исключение не обработано, программа немедленно останавливается, и Python выводит сообщение об ошибке (traceback).

```python
number = int("текст")   # ValueError
```

### 2. Конструкция try/except

Позволяет «перехватить» ошибку и обработать её без остановки программы:

```python
try:
    number = int(input("Введите число: "))
    print(100 / number)
except ValueError:
    print("Нужно ввести именно число")
except ZeroDivisionError:
    print("На ноль делить нельзя")
```

### 3. Обработка нескольких типов исключений одним блоком

```python
try:
    result = 10 / int(input("Введите число: "))
except (ValueError, ZeroDivisionError) as e:
    print(f"Произошла ошибка: {e}")
```

### 4. Блок else и finally

- `else` выполняется, только если в блоке `try` не возникло ошибок.
- `finally` выполняется **всегда**, независимо от того, была ошибка или нет — часто используется для «уборки» ресурсов (например, закрытия файла).

```python
try:
    number = int(input("Введите число: "))
except ValueError:
    print("Некорректный ввод")
else:
    print(f"Ты ввёл число {number}")
finally:
    print("Обработка ввода завершена")
```

### 5. Самые частые встроенные исключения

| Исключение | Когда возникает |
|---|---|
| `ValueError` | некорректное значение (например, `int("abc")`) |
| `TypeError` | операция применена к несовместимому типу |
| `ZeroDivisionError` | деление на ноль |
| `IndexError` | обращение к несуществующему индексу списка |
| `KeyError` | обращение к несуществующему ключу словаря |
| `FileNotFoundError` | попытка открыть несуществующий файл |

### 6. Практический пример: функция безопасного деления

```python
def safe_divide(a, b):
    try:
        return a / b
    except ZeroDivisionError:
        return "Ошибка: деление на ноль"
    except TypeError:
        return "Ошибка: некорректный тип данных"

print(safe_divide(10, 2))     # 5.0
print(safe_divide(10, 0))     # Ошибка: деление на ноль
print(safe_divide(10, "a"))   # Ошибка: некорректный тип данных
```

### Резюме урока

- `try/except` позволяет обрабатывать ошибки без остановки выполнения программы.
- `else` выполняется при отсутствии ошибок, `finally` — выполняется всегда.
- Обработка исключений делает программы значительно надёжнее при работе с пользовательским вводом, файлами и внешними данными.
