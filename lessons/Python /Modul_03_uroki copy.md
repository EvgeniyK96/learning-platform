## Урок 3.1. Условный оператор if / elif / else
**Время: 45 минут**

### Цель урока
Научиться управлять логикой программы в зависимости от выполнения условий с помощью конструкции `if / elif / else`.

### 1. Базовая конструкция if

Условный оператор позволяет выполнять определённый блок кода только при выполнении заданного условия.

```python
age = 20

if age >= 18:
    print("Доступ разрешён")
```

Обрати внимание на **двоеточие** после условия и **отступ** (4 пробела) перед кодом внутри блока `if`. Отступы в Python — это не просто стиль, а часть синтаксиса: именно они определяют, какой код относится к какому блоку.

### 2. Конструкция if / else

```python
age = 15

if age >= 18:
    print("Доступ разрешён")
else:
    print("Доступ запрещён")
```

### 3. Конструкция if / elif / else

Когда нужно проверить несколько условий по очереди, используется `elif` (сокращение от «else if»):

```python
score = 75

if score >= 90:
    grade = "A"
elif score >= 75:
    grade = "B"
elif score >= 60:
    grade = "C"
else:
    grade = "F"

print(f"Твоя оценка: {grade}")
```

Условия проверяются последовательно сверху вниз, и выполняется код только первого истинного условия — остальные проверки после этого пропускаются.

### 4. Вложенные условия

Условия можно вкладывать друг в друга:

```python
age = 25
has_ticket = True

if age >= 18:
    if has_ticket:
        print("Проходите в зал")
    else:
        print("Сначала купите билет")
else:
    print("Вход только с 18 лет")
```

Часто такую вложенность можно упростить с помощью логического оператора `and`:

```python
if age >= 18 and has_ticket:
    print("Проходите в зал")
```

### 5. Тернарный (краткий) условный оператор

Для простых случаев Python позволяет записать условие в одну строку:

```python
age = 20
status = "взрослый" if age >= 18 else "несовершеннолетний"
print(status)
```

### 6. Типичные ошибки новичков

- Забывают двоеточие `:` после условия.
- Путают отступы (смешивают пробелы и табуляцию).
- Используют `=` вместо `==` внутри условия.

```python
if age = 18:   # ошибка! нужно ==
    print("...")
```

### Резюме урока

- `if` выполняет код только при истинном условии, `else` — при ложном.
- `elif` позволяет проверить несколько условий по очереди.
- Отступы — обязательная часть синтаксиса Python, а не просто стиль.
- Для простых случаев можно использовать тернарный оператор в одну строку.

## Урок 3.2. Цикл while
**Время: 40 минут**

### Цель урока
Понять принцип работы цикла `while`, научиться писать циклы с условием выхода и избегать бесконечных циклов.

### 1. Как работает цикл while

Цикл `while` выполняет блок кода до тех пор, пока указанное условие остаётся истинным.

```python
count = 1

while count <= 5:
    print(count)
    count += 1
```

Этот код выведет числа от 1 до 5. Важно: переменная `count` должна изменяться внутри цикла — иначе условие никогда не станет ложным, и цикл будет выполняться бесконечно.

### 2. Бесконечные циклы

```python
# ОПАСНО! Условие никогда не станет False
while True:
    print("Бесконечно...")
```

Бесконечные циклы иногда используются намеренно (например, в играх или серверных программах), но требуют явного механизма выхода — например, через `break`.

### 3. Оператор break

`break` немедленно прерывает выполнение цикла, даже если условие всё ещё истинно:

```python
count = 1

while True:
    print(count)
    count += 1
    if count > 5:
        break
```

### 4. Оператор continue

`continue` пропускает оставшуюся часть текущей итерации и переходит к следующей проверке условия:

```python
count = 0

while count < 10:
    count += 1
    if count % 2 == 0:
        continue    # пропускаем чётные числа
    print(count)
```

Этот код выведет только нечётные числа от 1 до 9.

### 5. Практический пример: игра «Угадай число»

```python
secret_number = 42
guess = 0

while guess != secret_number:
    guess = int(input("Угадай число от 1 до 100: "))
    if guess < secret_number:
        print("Больше!")
    elif guess > secret_number:
        print("Меньше!")

print("Ты угадал!")
```

### 6. Типичные ошибки новичков

- Забывают изменить переменную-счётчик внутри цикла → бесконечный цикл.
- Путают `break` (полностью выходит из цикла) и `continue` (пропускает только текущую итерацию).

### Резюме урока

- Цикл `while` выполняется, пока условие истинно.
- Обязательно нужно предусмотреть, как условие станет ложным, чтобы избежать бесконечного цикла.
- `break` прерывает цикл полностью, `continue` — переходит к следующей итерации.

## Урок 3.3. Цикл for и функция range()
**Время: 45 минут**

### Цель урока
Научиться использовать цикл `for` для перебора последовательностей и работать с функцией `range()` для генерации числовых диапазонов.

### 1. Как работает цикл for

Цикл `for` в Python перебирает элементы любой последовательности (строки, списка, диапазона чисел и т.д.) по очереди:

```python
for letter in "Python":
    print(letter)
```

Этот код выведет каждую букву слова «Python» на отдельной строке.

### 2. Функция range()

`range()` создаёт последовательность чисел и часто используется вместе с `for`, когда нужно повторить действие определённое количество раз:

```python
for i in range(5):
    print(i)
# выведет: 0 1 2 3 4
```

Важно: `range(5)` генерирует числа от `0` до `4` включительно — то есть 5 чисел, но не включает само число 5.

`range()` может принимать до трёх аргументов:

```python
range(start, stop, step)
```

```python
for i in range(2, 10, 2):
    print(i)
# выведет: 2 4 6 8
```

### 3. Перебор списков через for

```python
fruits = ["яблоко", "банан", "груша"]

for fruit in fruits:
    print(fruit)
```

### 4. Получение индекса при переборе — функция enumerate()

```python
fruits = ["яблоко", "банан", "груша"]

for index, fruit in enumerate(fruits):
    print(index, fruit)
# 0 яблоко
# 1 банан
# 2 груша
```

### 5. Практический пример: таблица умножения

```python
number = int(input("Введите число: "))

for i in range(1, 11):
    print(f"{number} x {i} = {number * i}")
```

### 6. Разница между for и while

`for` используется, когда заранее известно количество повторений или есть последовательность для перебора. `while` — когда повторение зависит от условия, которое неизвестно заранее (например, «пока пользователь не угадает число»).

### Резюме урока

- Цикл `for` перебирает элементы последовательности: строки, списка, диапазона чисел.
- `range(start, stop, step)` генерирует числовую последовательность, не включая значение `stop`.
- `enumerate()` позволяет получить индекс элемента вместе с его значением при переборе.

## Урок 3.4. Вложенные циклы, break, continue, else в циклах
**Время: 45 минут**

### Цель урока
Научиться работать с вложенными циклами и понять дополнительные возможности управления циклами в Python.

### 1. Вложенные циклы

Один цикл можно поместить внутрь другого — это называется вложенным циклом. Часто используется для работы с таблицами и матрицами:

```python
for i in range(3):
    for j in range(3):
        print(f"i={i}, j={j}")
```

Внутренний цикл полностью выполняется для каждой итерации внешнего цикла.

### 2. Практический пример: треугольник из звёздочек

```python
height = 5

for i in range(1, height + 1):
    print("*" * i)
```

Результат:
```
*
**
***
****
*****
```

### 3. break во вложенных циклах

Важно помнить: `break` прерывает только тот цикл, в котором он находится напрямую, а не все циклы сразу:

```python
for i in range(3):
    for j in range(3):
        if j == 1:
            break        # прервёт только внутренний цикл
        print(i, j)
```

### 4. Конструкция else в циклах

В Python у циклов `for` и `while` есть необычная возможность — блок `else`, который выполняется, если цикл завершился **без** прерывания через `break`:

```python
for number in range(2, 10):
    if number % 7 == 0:
        print("Найдено число, делящееся на 7")
        break
else:
    print("Подходящих чисел не найдено")
```

Такая конструкция удобна, например, при поиске элемента в последовательности.

### 5. Практический пример: поиск простого числа

```python
number = int(input("Введите число: "))

for i in range(2, number):
    if number % i == 0:
        print(f"{number} составное")
        break
else:
    print(f"{number} простое")
```

### 6. Типичные ошибки новичков

- Ожидают, что `break` во внутреннем цикле прервёт и внешний цикл тоже — это не так.
- Забывают, что блок `else` в цикле относится именно к циклу, а не к предыдущему `if`.

### Резюме урока

- Вложенные циклы позволяют работать с двумерными структурами данных, такими как таблицы и матрицы.
- `break` прерывает только тот цикл, внутри которого он написан напрямую.
- Блок `else` при цикле выполняется, только если цикл завершился без `break`.

## Урок 3.5. Практика: решение задач с циклами и условиями
**Время: 60 минут**

### Цель урока
Закрепить материал модуля на практике, решив несколько классических задач с использованием условий и циклов.

### 1. Задача 1. Определение чётности чисел от 1 до 20

```python
for number in range(1, 21):
    if number % 2 == 0:
        print(f"{number} — чётное")
    else:
        print(f"{number} — нечётное")
```

### 2. Задача 2. Сумма чисел от 1 до N

```python
n = int(input("Введите N: "))
total = 0

for i in range(1, n + 1):
    total += i

print(f"Сумма чисел от 1 до {n}: {total}")
```

### 3. Задача 3. Подсчёт гласных букв в слове

```python
word = input("Введите слово: ")
vowels = "аеёиоуыэюя"
count = 0

for letter in word.lower():
    if letter in vowels:
        count += 1

print(f"Количество гласных: {count}")
```

### 4. Задача 4. FizzBuzz

Классическая задача, часто встречающаяся на технических собеседованиях:

```python
for number in range(1, 101):
    if number % 3 == 0 and number % 5 == 0:
        print("FizzBuzz")
    elif number % 3 == 0:
        print("Fizz")
    elif number % 5 == 0:
        print("Buzz")
    else:
        print(number)
```

### 5. Задача 5. Таблица умножения от 1 до 5

```python
for i in range(1, 6):
    for j in range(1, 6):
        print(f"{i} x {j} = {i * j}")
    print("---")
```

### 6. Как подходить к решению подобных задач

1. Внимательно прочитай условие и определи, что является входными, а что выходными данными.
2. Реши задачу «на бумаге» или словами, прежде чем писать код.
3. Определи, нужен ли цикл `for` (известное количество повторений) или `while` (повторение по условию).
4. Напиши базовую версию кода и протестируй на нескольких примерах.
5. Проверь пограничные случаи (например, N = 0, пустая строка, отрицательные числа).

### Резюме урока

- Практика — ключевой этап закрепления любой новой темы в программировании.
- Разбор условия задачи на «входные данные → обработка → результат» помогает выстроить логику решения.
- Умение комбинировать циклы и условия — база для решения большинства практических задач в Python.
