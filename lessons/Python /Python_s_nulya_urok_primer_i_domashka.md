### 1. Что такое переменная

Переменная — это «именованная коробка» в памяти компьютера, в которую можно положить какое-то значение, а потом обращаться к этому значению по имени.

В Python переменную не нужно заранее объявлять с указанием типа (как в некоторых других языках) — достаточно просто присвоить ей значение с помощью знака `=`.

```python
name = "Алина"
age = 19
city = "Алматы"
```

Здесь `name`, `age`, `city` — это имена переменных, а `"Алина"`, `19`, `"Алматы"` — их значения.

Python сам определяет тип значения в момент присваивания. Это называется **динамической типизацией**.

---

### 2. Правила именования переменных

- Имя может состоять из букв, цифр и знака подчёркивания `_`.
- Имя не может начинаться с цифры.
- Python чувствителен к регистру: `age` и `Age` — это разные переменные.
- Нельзя использовать зарезервированные слова языка (`if`, `for`, `class` и т.д.).
- Договорённость сообщества (PEP 8): имена переменных пишутся в стиле `snake_case`, например `user_name`, а не `userName`.

```python
# Плохо
UserAge = 20
2age = 20        # ошибка: имя не может начинаться с цифры

# Хорошо
user_age = 20
```

---

### 3. Основные типы данных

В Python есть несколько базовых («встроенных») типов данных, с которыми ты будешь работать постоянно:

| Тип | Название в Python | Пример | Описание |
|-----|--------------------|--------|----------|
| Целое число | `int` | `10`, `-5`, `1000` | Числа без дробной части |
| Дробное число | `float` | `3.14`, `-0.5` | Числа с плавающей точкой |
| Строка | `str` | `"Привет"`, `'мир'` | Текстовые данные |
| Логический тип | `bool` | `True`, `False` | Истина/ложь |

Проверить тип переменной можно с помощью встроенной функции `type()`:

```python
name = "Алина"
age = 19
height = 1.65
is_student = True

print(type(name))        # <class 'str'>
print(type(age))         # <class 'int'>
print(type(height))      # <class 'float'>
print(type(is_student))  # <class 'bool'>
```

---

### 4. Изменение значения переменной

Переменная может менять и значение, и тип в процессе работы программы:

```python
x = 5        # x - целое число
print(x)     # 5

x = "теперь я строка"
print(x)     # теперь я строка
```

Такое поведение — особенность динамической типизации Python. Оно удобно, но требует аккуратности: важно всегда понимать, какой тип данных сейчас хранится в переменной.

---

### 5. Множественное присваивание

Python позволяет присваивать значения сразу нескольким переменным в одной строке:

```python
a, b, c = 1, 2, 3

# Или присвоить одно и то же значение нескольким переменным
x = y = z = 0
```

---

### 6. Типичные ошибки новичков

- **Использование переменной до её создания.** Python выдаст `NameError`, если переменная ещё не была объявлена.
- **Опечатки в имени переменной** — `usr_name` вместо `user_name` создаст новую переменную, а не обратится к старой.
- **Путаница между строкой и числом**: `"5" + "3"` даст строку `"53"`, а не число `8`, потому что обе переменные — строки.

```python
a = "5"
b = "3"
print(a + b)      # 53 (склеивание строк, а не сложение чисел)

a = 5
b = 3
print(a + b)      # 8 (сложение чисел)
```

---

### 7. Краткое резюме урока

- Переменная — это имя, привязанное к значению в памяти.
- Тип переменной определяется автоматически в момент присваивания.
- Основные типы: `int`, `float`, `str`, `bool`.
- Тип переменной можно проверить функцией `type()`.
- Имена переменных пишутся в `snake_case` и должны быть понятными.
